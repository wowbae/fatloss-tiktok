# 📱 Mobile-First Landing Page: Книга по похудению
**Стиль:** Apple Minimalistic / High-Typography
**Целевая аудитория:** Азербайджан (Баку)
**Технологический стек:** HTML5, Tailwind CSS, Vanilla JS, Bun (Сборщик)

---

## 📄 1. СТРУКТУРА И ТЕКСТ ПО ЭКРАНАМ (100vh каждый)

### Экран 1: Главный (Крючок / First Impression)
* **Визуальное решение:** Сплит-экран (фото блогера «До / После»). Нижняя треть экрана плавно затемняется (black gradient), чтобы текст оставался абсолютно читаемым.
* **Заголовок (Шрифт: Inter/San Francisco, Bold, XL):** `Минус 12 кг без голодовок и срывов.`
* **Подзаголовок (Светло-серый, Regular):** `Пошаговое руководство по похудению от [Имя Блогера].`
* **Кнопка (CTA) (Скругленная, Apple-style, акцентный цвет):** `Получить книгу за [Цена] AZN ➔`

### Экран 2: Проявление боли (Emotional Trigger)
* **Визуальное решение:** Глубокий черный фон (#000000). Максимальный фокус на типографике. Огромные буквы, много свободного пространства.
* **Заголовок:** `Диеты не работают.`
* **Тезисы (Появляются поочередно при скролле с эффектом fade-in):**
    * *«Жесткие ограничения всегда ведут к срывам.»*
    * *«Спортзалы отнимают слишком много времени.»*
    * *«После голодания вес всегда возвращается назад.»*

### Экран 3: Презентация решения (The Product)
* **Визуальное решение:** Переход на чистый белый фон (#FFFFFF). В центре — минималистичный 3D-рендер обложки книги с мягкой премиальной тенью.
* **Заголовок:** `Ваша новая система.`
* **Текст (Крупный, с хорошим межстрочным интервалом):** `Это не просто книга. Это четкий пошаговый алгоритм, полностью адаптированный под наш образ жизни и локальную кухню. Никаких экзотических продуктов. Только то, что есть в любом супермаркете Баку.`

### Экран 4: Ценность и наполнение (Features)
* **Визуальное решение:** Мягкий серый фон (#F5F5F7). Чистая сетка (одна карточка под другой) с тонкими разделителями.
* **Заголовок:** `Внутри руководства:`
* **Пункты:**
    * 📋 **30 готовых рецептов** на каждый день из доступных продуктов.
    * 🧠 **Психология привычек:** как навсегда победить тягу к сладкому.
    * 🔄 **Стабильный результат:** метод удержания веса годами.

### Экран 5: Финальный призыв (Action & Scarcity)
* **Визуальное решение:** Черный фон. Финальное портретное фото улыбающегося блогера, излучающего здоровье и уверенность.
* **Заголовок:** `Начните сегодня.`
* **Подзаголовок:** `Цена изменится через 24 часа. Сделайте главную инвестицию в свое тело.`
* **Кнопка (CTA):** `Купить книгу за [Цена] AZN`

## 🛠️ 2. АРХИТЕКТУРА И ТЕХНОЛОГИЧЕСКИЙ СТЕК

Проект собирается без использования тяжелых JS-фреймворков (No React/Vue) для достижения 100 баллов в Google Lighthouse и мгновенной загрузки на мобильных устройствах.

```text
├── /dist                  # Минифицированный продакшн-код (автогенерация)
│   ├── index.html
│   └── /assets
├── /src                   # Исходный код
│   ├── /assets            # Изображения блогера и книги (строго в WebP)
│   ├── index.html         # Семантическая разметка
│   └── input.css          # Директивы Tailwind
├── tailwind.config.js     # Настройки шрифтов и кастомных цветов
├── package.json
└── bun.lockb
```

## Инструменты оптимизации:

- Tailwind CSS — для верстки «на лету» без раздувания CSS-файлов.

- Intersection Observer API — встроенный в браузер инструмент для плавного появления текста Apple-style при скролле (без внешних библиотек).
