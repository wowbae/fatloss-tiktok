Quick Start

A Simple MCP Server

Define a tool in Python with just a few lines

weather_server.py
1from mcp import Server, Tool 2 3app = Server("weather") 4 5@app.tool() 6async def get_weather(city: str) -> str: 7 """Get current weather for a city.""" 8 data = await fetch_weather(city) 9 return f"{data.temp}°F, {data.condition}" 10 11app.run()

The AI discovers get_weather automatically — no manual wiring
